# revise-lite

Offline study note summarizer that converts raw lecture notes into revision-ready sheets instantly.

## Problem

Students accumulate large volumes of notes but struggle to convert them into structured revision material before exams.

revise-lite extracts key sentences and keywords automatically.

## Who It Helps

- University students
- Cybersecurity learners
- Researchers
- Self-learners
- Exam candidates

## Features

- Fully offline processing
- Key sentence extraction
- Keyword frequency detection
- Instant revision sheet generation
- Zero dependencies

## Installation

git clone https://github.com/Syedsaadhhh/revise-lite

cd revise-lite

python main.py examples/sample_notes.txt

## Why This Matters

Reduces revision preparation time dramatically before exams.
